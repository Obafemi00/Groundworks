# Founder Groundworks | Brand Style Guide (v1.0)

## Color System

### Primary Colors
- **FG Deep Navy**: `#0E1F36` - Primary backgrounds, header sections, institutional materials
- **FG Pillar White**: `#FFFFFF` - Clean backgrounds, negative space
- **FG Heritage Gold**: `#C4A463` - Accent lines, icons, highlights (use sparingly - max 5%)

### Secondary Colors
- **FG Slate Grey**: `#5A6473` - Secondary text, dividers, muted UI
- **FG Soft Sand**: `#E9E4DB` - Light background panels, cards, supporting sections
- **FG Midnight Blue**: `#081524` - Depth sections, overlays

### Usage Ratio
- Navy: 60%
- White/Sand: 35%
- Gold: 5% (maximum, for accent only)

## Typography

### Headings: Cormorant Garamond
- Used for: H1, H2, large titles, premium statements
- Weights: Medium (500), Semibold (600)
- Letter spacing: -0.01em
- Line height: 1.2

### Body & UI: Inter Variable
- Used for: body text, paragraphs, UI labels, buttons, captions
- Body: 16-18px
- Small text: 12-14px
- Buttons: Inter Semibold (600)

## Button Specifications

### Primary Button
- Background: FG Deep Navy (`#0E1F36`)
- Text: White
- Border radius: 4px
- Padding: 14px 28px (vertical horizontal)
- Typeface: Inter Semibold

### Secondary Button
- Background: White
- Border: 1px FG Deep Navy
- Text: FG Deep Navy
- Border radius: 4px
- Padding: 14px 28px

### Accent Button (use sparingly)
- Background: Heritage Gold (`#C4A463`)
- Text: FG Deep Navy
- Border radius: 4px
- Padding: 14px 28px

## Spacing

- Section padding: 80-120px (desktop), 48px (mobile)
- Internal component padding: 20-24px minimum
- Container max-width: 1200px

## Logo Usage

- Primary logo: Logo icon + wordmark
- Approved backgrounds: FG Deep Navy or White
- Do not: Alter proportions, apply drop shadows/gradients/textures, place on busy imagery
